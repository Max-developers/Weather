# Weather

**Во избежание ошибок, убедитесь, что у вас установлен модуль requests. 
Если не установлен, вы можете установить его с помощью следующей команды:**

```
pip install requests
```

Как пользоваться приложением:
+ Получите API ключ для работы с погодным API, например, OpenWeatherMap (https://openweathermap.org/api).
+ Замените строку API_KEY = 'ваш_api_key' на ваш реальный API ключ.
+ Запустите приложение.

```
python3 setup.py
```

+ Теперь вы можете ввести название города, и оно вернет текущую погоду в этом городе.

Примечания:
+ Убедитесь, что вы подключены к интернету, чтобы получить данные о погоде.
+ Программа обрабатывает только ошибки, возникающие при неверном введении города. 
+ Убедитесь, что вводите названия городов корректно.
